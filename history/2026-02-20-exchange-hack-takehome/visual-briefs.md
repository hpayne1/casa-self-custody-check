# Visual briefs (designer handoff)

## X post 1 visual (recommended: no visual)

Crisis single posts default to no visual unless a checklist card materially improves clarity. The text-only format reads cleanly on X and avoids sensational framing.

## X post 2 thread visual (recommended: checklist card)

* **Recommendation**: checklist card
* **Purpose**: make the thread saveable/shareable; provide a calm plan
* **On-image copy**
  * **Headline**: After exchange headlines: a calm 4-step plan
  * **Steps**:
    1. Spend vs long-term
    2. Start small
    3. Test recovery
    4. Reduce single points of failure
* **Layout notes**: numbered steps, neutral shapes/icons; no breaking-news styling
* **Specs**: 1600x900 (primary), 1200x675, 1080x1080

## Blog hero visual

* **Purpose**: set first-class tone; avoid sensational framing
* **Concept**: "Access vs ownership" — two-column comparison with neutral shapes
* **Optional on-image copy**: After an exchange hack: first-aid now, a custody plan next
* **Do**: minimal, calm, high-contrast, readable typography
* **Don't**: "breaking news" banners, red alert palettes, shaming screenshots
* **Specs**: 1200x630 (blog/social)
